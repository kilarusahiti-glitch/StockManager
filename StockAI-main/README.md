# StockManager - AI-Powered Stock Analysis & Management App

StockManager is a web application that helps users **analyze, track, and manage stock investments** using AI insights. It provides real-time stock data, investment suggestions, and a simple interface to buy/sell stocks and monitor portfolio performance.

## 🛠️ Features

- **Real-time stock data** from NSE
- **AI-based stock recommendations**
- **Buy/Sell simulation** for investment tracking
- **User authentication** using Firebase
- **Portfolio management dashboard**
- **Responsive frontend** built with HTML/CSS/JS
- **Backend API** using Flask
- **Data storage** with Firebase Firestore

## 🏗️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python, Flask  
- **Database:** Firebase Firestore  
- **AI/Analytics:** Python (Data Analysis, yFinance API)  
- **Hosting/Deployment:** Render, Firebase
